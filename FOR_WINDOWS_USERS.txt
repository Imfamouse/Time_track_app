🚀 САМЫЙ ПРОСТОЙ СПОСОБ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Скачайте/клонируйте проект на свой Windows компьютер

2. Установите Docker Desktop:
   https://www.docker.com/products/docker-desktop/
   (5 минут)

3. Дважды кликните на:
   WINDOWS_QUICKSTART.bat

4. Откройте браузер:
   http://localhost:5173

🎉 ГОТОВО! Приложение работает!


📋 АЛЬТЕРНАТИВНЫЙ СПОСОБ (БЕЗ DOCKER):

Если не хотите устанавливать Docker:

1. Установите:
   • Python 3.12+ (с галочкой "Add to PATH"!)
   • Node.js 20+

2. Дважды кликните:
   ▶️  start-backend.bat   (откроется первое окно)
   ▶️  start-frontend.bat  (откроется второе окно)

3. Откройте: http://localhost:5173
